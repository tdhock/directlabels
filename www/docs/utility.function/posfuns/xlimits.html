directlabels - utility.function - Positioning Method - xlimits

Return the positions of the plot horizontal limits in cm, for use as the limit argument to qp.labels.

xlimits <- function(...){
  convertX(unit(c(0,1),"npc"),"cm",valueOnly=TRUE)
}
